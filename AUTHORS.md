## Acknowledgements

This methodology builds on two prior bodies of work (both cited in the guide's provenance note):

- **Anthropic's official Agent Skills best-practices** — the single-skill evaluation concepts (baseline-without-skill, evaluations as source of truth, multi-model testing on Haiku / Sonnet / Opus).
- **[`superpowers`](https://github.com/obra/superpowers)** by Jesse Vincent (`@obra`) — the writing-skills methodology, in particular the "NO SKILL WITHOUT A FAILING TEST FIRST" rule (RED → GREEN → REFACTOR applied to skills) and the rationalization-resistance framework that informed the `posture & responsibility` dimension.

The extension of these ideas to the *system* level — trigger disambiguation, coverage gaps, cross-agent contracts, system-wide regression, end-to-end orchestration — is this project's own contribution.
